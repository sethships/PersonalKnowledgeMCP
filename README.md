# Personal Knowledge MCP

A personal RAG (Retrieval-Augmented Generation) knowledgebase system built on the Model Context Protocol (MCP), designed to manage multiple knowledge stores with flexible deployment options.

## Overview

Personal Knowledge MCP is a containerized knowledge management system that:

- Supports multiple database and storage backends
- Provides intelligent navigation and retrieval across heterogeneous knowledge sources
- Deploys locally or to cloud infrastructure using Kubernetes
- Integrates seamlessly with AI systems via the Model Context Protocol
